> 所以devicetype 對應到 一個device template?

# AI Say
對，沒錯。`DeviceTemplate` 這張表有唯一的約束來確保這件事：

```java
@Table(name = "device_templates",
    uniqueConstraints = @UniqueConstraint(columnNames = { "tenant_id", "device_type" }))
```

意思是 **每個租戶底下，一種 `device_type` 只能有一個 `DeviceTemplate`**。

所以關聯是：

```
DeviceTemplate   ───1─────────   Device.device_type
(每種 type 只有一份 schema)    (多個 device 共用同一種 type)
```

舉例來說：

| device_type | DeviceTemplate (schema 定義) | 對應的 Device 實例 |
|---|---|---|
| `STREET_LIGHT` | `{ "brightness": "number", "bulbType": "string", ... }` | 忠孝東路1號燈、仁愛路2號燈... |
| `AIR_QUALITY` | `{ "pm25": "number", "co2": "number", ... }` | 松山站測站、信義區測站... |

**一個 device_type 對應一份 JSON Schema（DeviceTemplate），而多個 Device 實例可以共用同一種 type。**
