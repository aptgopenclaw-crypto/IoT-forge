## 🔑 關鍵組件對照表

| 層次 | 組件 | 您的程式碼實例 | 核心作用 |
|------|------|----------------|----------|
| **啟動引擎** | Auto-configuration | `@EnableAsync` | 自動配置非同步執行緒池 |
| **核心引擎** | IoC 容器 | `@Service`, `@Bean` | 管理 Bean 生命週期 |
| **核心引擎** | DI 依賴注入 | `@RequiredArgsConstructor` | 自動注入 Repository |
| **擴展引擎** | AOP 切面 | `BaseLoggerAspect` | 攔截審計日誌 |
| **擴展引擎** | 動態代理 | `@Async`, `@RunInSystemTenantContext` | 生成代理物件 |
| **擴展引擎** | Web 引擎 | `DispatcherServlet` | 處理 HTTP 請求 |
| **擴展引擎** | 事務管理 | `@Transactional` (隱式) | 管理資料庫事務 |
